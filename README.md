# CareQuest System Architecture

System architecture for AI-native neurodivergence care workforce training.

## Purpose

This repository maps how CareQuest connects:

- gamified learning
- behavioral simulations
- AI supervision
- competency scoring
- certification readiness
- workforce graph
- placement infrastructure

## Core System Thesis

The future of care workforce training will require more than content delivery.

It will need an integrated infrastructure layer that can:

1. onboard learners
2. personalize learning paths
3. simulate behavioral scenarios
4. evaluate competency
5. track progression
6. verify readiness
7. connect learners to jobs

## Core Modules

- Learner Onboarding Engine
- Mission-Based Learning Engine
- Simulation Engine
- AI Supervisor
- Competency Engine
- Certification Readiness Engine
- Workforce Graph
- Employer / Training Provider Dashboard

ABAT/RBT learner completion and readiness.
